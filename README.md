# Framie

Framie is an iframe widget SDK monorepo.

## Packages

- `@framie/core`: framework-agnostic widget engine and messaging channel.
- `@framie/react`: React bindings on top of core.
- `@framie/peer`: helper channel for code running inside iframe.

## Quick Start

```bash
npm install
npm run build
```

## Workspace Commands

```bash
npm run build
npm run typecheck
npm run dev
```
